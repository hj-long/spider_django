#### 后端部分：spider_django

下载\克隆项目：

git clone https://github.com/hj-long/spider_django

第一步：要安装django,这里使用的是django==2.1.5版本

```
pip install django==2.1.5

```

第二步：启动项目

cd my_splider
```
python manage.py runserver 8000

```
